JavaScript Basics

This is a sample webpage that calls functions from js files. You may view the code on your local machines

Conditional Statements

if | Write a letter and check if it's a vowel.

if-else | Input two numbers and check if one of the number is 100 or if the sum of the two numbers is 100

if-else-if | Input a day and check if it's weekend

switch | Input a month and check its season

ternary? | Input a word and check if it's a palindrome

LOOPS

for | Input a number and print its Multiplication Table

do-while | Input a number and get its factorial

while | Input a number and reverse it

Basic Exercises

Number of Vowels | Input a word or phrase and count the number of vowels

Largest Number | Input series of numbers separated by comma(,) and get the largest number

Sign of Product | Input q series of numbers separated by comma(,) and check if the product is positive or negative

Print Odd | Input a series of numbers separated by comma(,) and print all the odd numbers from the list

Print Even | Input a series of numbers separated by comma(,) and print all the even numbers from the list

FizzBuzz | Input a series of numbers separated by comma(,) and print Fizz if divisible by 3, Buzz if divisible by 5, and FizzBuzz if divisible by both

Remove Duplicates | Input a series of numbers separated by comma(,) and remove duplicates
